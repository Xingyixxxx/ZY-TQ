## 功能特性

- **🪙 3D 铜钱抛掷动画** — 逼真的乾隆通宝 3D 翻滚，正面/背面清晰区分
- **⚖️ 标准三枚铜钱法** — 严格依据易经规则计算老阴 (6)、少阳 (7)、少阴 (8)、老阳 (9)
- **📖 六十四卦完整数据** — 所有 64 卦均收录卦名、卦辞、彖传与解释
- **🔄 本卦 + 变卦** — 自动识别动爻，推演变卦
- **✨ 沉浸式体验** — 卦象虚化后卦辞逐渐浮现，具有仪式感
- **🎇 背景粒子系统** — 金色粒子缓缓升腾的环境氛围

## 如何运行

本项目为纯静态网页，**无需任何构建工具或后端**，直接用浏览器打开即可。

### 方法一：Python 本地服务器（推荐）

```bash
# 进入项目目录
cd /path/to/ZY

# Python 3
python3 -m http.server 8080
```

打开浏览器访问 → `http://localhost:8080`

### 方法二：VS Code Live Server

1. 用 VS Code 打开项目文件夹
2. 安装 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 插件
3. 右键 `index.html` → **Open with Live Server**

### 方法三：直接双击

直接双击 `index.html` 在浏览器中打开（部分浏览器可能因跨域限制导致字体加载失败，推荐使用方法一）。

## 自定义指南

### 修改卦辞解释

编辑 `hexagramData.js` 中的 `HEXAGRAM_DATA` 对象：

```js
// 格式：'二进制编码': ['卦名', '卦辞', '解释']
'111111': ['乾为天', '天行健，君子以自强不息。', '...详细解释...'],
```

**编码规则：** 6 位二进制字符串，代表从**初爻（下）→ 上爻（上）**的阴阳顺序：
- `1` = 阳爻 ━━━
- `0` = 阴爻 ━ ━

例如：乾卦（六爻全阳）= `'111111'`，坤卦（六爻全阴）= `'000000'`

### 修改界面文字

- **标题 / 副标题**：修改 `index.html` 中的 `<h1>` 和 `.subtitle`
- **页脚文字**：修改 `<footer>` 内容

### 修改视觉风格

编辑 `style.css` 顶部的 CSS 变量：

```css
:root {
    --gold:        #d4a94a;  /* 主色调（金色）*/
    --gold-light:  #f0c96e;  /* 高亮金色 */
    --bg-dark:     #0a0a0f;  /* 背景深色 */
    --bg-panel:    #12121a;  /* 面板背景 */
}
```

### 修改铜钱动画

在 `script.js` 的 `animateCoins()` 函数中：

```js
const flipCount    = 3 + Math.floor(Math.random() * 4); // 翻转圈数 3~6 圈
const peakHeight   = -200 - Math.random() * 100;        // 抛起高度（负值=向上）
const duration     = 1400 + index * 100;                // 动画时长（毫秒）
```

## 使用说明

1. 打开网页，静心片刻，**默念所问之事**
2. 点击 **"开始起卦"** 按钮
3. 观看 3D 铜钱翻滚，系统自动记录当前爻（第一爻 = 初爻，自下而上）
4. **重复点击 6 次**，依次生成六爻
5. 六爻排定后，系统自动计算本卦与变卦
6. 稍后卦象渐隐，**卦辞解释**缓缓浮现
7. 阅读完毕后点击 **"重新起卦"** 开始下一次问卜

## 易经起卦规则说明

| 三枚铜钱之和 | 爻名 | 阴/阳 | 是否动爻 |
|:-----------:|:----:|:-----:|:-------:|
| 6 (字字字)  | 老阴  | 阴爻  | ✅ 变阳  |
| 7 (背字字)  | 少阳  | 阳爻  | ❌       |
| 8 (背背字)  | 少阴  | 阴爻  | ❌       |
| 9 (背背背)  | 老阳  | 阳爻  | ✅ 变阴  |

> **字面（正面）= 2，背面 = 3**，三枚求和即得爻值。
